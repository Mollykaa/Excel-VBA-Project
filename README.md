## Overview

A collection of Excel workbooks built for the Business Analytics Modelling course at IMT Business School. The projects cover the full analytics workflow — from raw data management and pivot-table reporting, to VBA macro automation and statistical forecasting with Solver optimization.

### 1. Excel Data Analysis — Supplier Orders, Store Metrics & Customer Payments

Analyzes operational data across three business areas: supplier order tracking, store performance metrics, and customer payment records. Applies Excel formulas, conditional formatting, and summary dashboards to surface key business insights.

**Techniques:** VLOOKUP / XLOOKUP, IF / nested logic, conditional formatting, summary tables, KPI cards

---

### 2. Excel Data Management & Visualization — Tables, Filters & Pivot Analytics

Demonstrates structured data management using Excel Tables with dynamic ranges, advanced filtering, and PivotTable / PivotChart reporting. Built to simulate a real business reporting workflow.

**Techniques:** Excel Tables, AutoFilter, advanced filtering, PivotTables, PivotCharts, slicers

---

### 3. Statistical Forecasting & Optimization Models using Excel VBA and Solver

Applies quantitative methods to business forecasting and resource optimization. Uses Excel Solver to find optimal solutions under constraints, and VBA macros to automate scenario analysis.

**Techniques:** Moving averages, exponential smoothing, linear programming, Solver (GRG Nonlinear / Simplex LP), VBA automation

---

### 4. VBA Automation for Inventory & Store Management

Automates inventory tracking and store management workflows using VBA macros and UserForms. Reduces manual data entry and enforces business rules through coded logic.

**Techniques:** VBA macros, UserForms, loops, conditionals, worksheet events, data validation

---

### 5. VBA Automation for Review Analysis — Camera Products & AirBnB Rentals

Parses and scores customer review data for two product categories (camera products and AirBnB rentals) using VBA string functions and automated scoring logic.

**Techniques:** VBA string parsing, keyword extraction, automated scoring, loop-based data processing

---

### 6. VBA Automation for Review Analysis — Criteria Analysis & AirBnB Rating

Extends the review analysis framework with a multi-criteria rating model. Applies weighted scoring across review dimensions and generates automated summary reports.

**Techniques:** Multi-criteria decision analysis, weighted scoring models, VBA conditionals, automated reporting

## How to Open

```
1. Download the .xlsm file
2. Open in Microsoft Excel (2016 or later recommended)
3. Enable macros when prompted (required for VBA workbooks)
4. Navigate using the sheet tabs or any built-in navigation buttons
```

> **Note:** Files use `.xlsm` format (macro-enabled workbook). Macros must be enabled for VBA automation to function correctly.
